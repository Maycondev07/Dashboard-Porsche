# Projeto

  *Esse projeto foi desenvolvido em resultado do curso de Introdução a AI Reports.*

  Durante o curso aprendemos a desenvolver dashboards de uma forma mais agil e visualmente melhor do que a convencional, no qual o dashboard que produzimos é um projeto de análise de dados desenvolvido para transformar informações comerciais de vendas de veículos Porsche em uma experiência visual, interativa e orientada a decisões.

# Visão geral

  A aplicação apresenta um painel executivo no estilo BI, permitindo explorar indicadores de desempenho, padrões de vendas e comportamento dos clientes através de filtros e métricas estratégicas.

  O dashboard foi estruturado com uma interface moderna inspirada no universo premium da marca Porsche, utilizando uma identidade visual sofisticada para transmitir características como luxo, precisão e performance.
